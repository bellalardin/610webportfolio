WPTS Radio Website

The brand new website for WPTS Radio! I designed this site using the ProRadio WordPress theme. The site features weekly articles written by WPTS staffers and a variety of ways to interact with the station.

Hero Hand

The Hero Hand lets any user become a superhero in just a few seconds! The device houses a foam dart launching system, wearable by the user on their hand. The Hero Hand was designed with the goal of bringing superhero powers to life for children and making it easy for them to play projectile-launching games even if they had limited motor capabilities.

More information about the Hero Hand can be found here.

Home Pinball Controller

The Home Pinball Controller is an interactive user-interface for pinball. It provides physical, in-real-life feedback to the pinball bumpers as well as a realistic plunger that simulates an arcade pinball machine, making a far more immersive experience than simply pressing buttons on a keyboard.

More information about the Home Pinball Machine can be found here.

Scholardarity

ScholarDarity is a point-based leaderboard system in which college students submit academic tasks that they have completed throughout the day in order to earn points that go towards prizes.
